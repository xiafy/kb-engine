# ANCA Vasculitis

> version: 1.0 | source: sop.md v2.2 split

## Trial Design

### Control Arm Design
IF indication == "ANCA vasculitis GC-sparing":
→ Alternative design (investigational drug replaces prednisone), double-dummy to maintain blinding, both arms receive CYC/RTX background therapy

### Endpoint Selection
IF indication == "ANCA vasculitis":
→ Remission = BVAS=0 (adjudication committee assessment); sustained remission = BVAS=0 at both Wk26 and Wk52
→ Primary endpoint does not include GC dose threshold
→ Secondary endpoints include renal biomarkers (eGFR/UACR/MCP-1) + GTI + SF-36/EQ-5D-5L
