> version: 1.0 | source: sop.md v2.2 拆分

## Trial Design

### 对照组设计
IF 适应症 == "ANCA 血管炎 GC-sparing":
→ 替代设计（试验药替代 prednisone），双模拟确保盲法，两臂均接受 CYC/RTX 背景治疗

### 终点选择
IF 适应症 == "ANCA 血管炎":
→ 缓解 = BVAS=0（裁定委员会评估），sustained remission = Wk26+Wk52 均 BVAS=0
→ 主要终点不含 GC 剂量阈值
→ 次要终点含肾脏生物标志物（eGFR/UACR/MCP-1）+ GTI + SF-36/EQ-5D-5L
