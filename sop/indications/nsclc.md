# NSCLC

> version: 1.0 | source: sop.md v2.2 拆分

## Trial Design

### Biomarker 检测策略
IF 适应症 == "KRAS G12C 肿瘤": → 双伴随诊断（组织 PCR + 血浆 ctDNA NGS），与 NDA 同步提交 sPMA。

### 分层因子
NSCLC 加速审批推荐分层/亚组因子: EGFR/ALK 状态 + PD-L1 表达 + 既往治疗线数。
